# 📰 Fake News Detection Using NLP

A natural language processing project that detects and classifies 
fake news articles using machine learning and deep learning techniques.

## 📌 About The Project
Misinformation spreads faster than ever online. This project builds 
an AI model that classifies news articles as real or fake using NLP 
techniques — achieving 96% accuracy across 40,000+ articles.

## 🛠️ Tech Stack
- Python
- TensorFlow / Keras
- Scikit-learn
- NLTK
- Pandas
- NumPy
- Matplotlib

## 📊 Model Performance
- Accuracy: 96%
- Techniques: TF-IDF, Word Embeddings, LSTM, BERT
- Models Compared: Logistic Regression, LSTM, BERT
- Dataset: 40,000+ news articles

## ⚙️ How To Run

### 1. Clone the repo
git clone https://github.com/singireddyakanksha04-source/fake-news-detection.git

### 2. Install dependencies
pip install tensorflow scikit-learn nltk pandas numpy matplotlib

### 3. Run the model
python fake_news_detection.py
